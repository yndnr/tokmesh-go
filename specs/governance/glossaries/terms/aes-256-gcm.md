# AES-256-GCM

## 解释
与 AES-128-GCM 类似，但使用 **256 位** 的密钥长度。它提供了更高的理论安全强度（抗暴力破解能力），但相应地在 CPU 运算上会有轻微的性能开销（现代 CPU 上差异通常小于 20%）。也是一种 AEAD 算法。

## 使用场景
- **高安全级别应用**: 银行、军事等级的数据传输，对长期安全性有极高要求的场景。
- **抗量子准备**: 许多专家认为 256 位密钥在面对未来量子计算机攻击时比 128 位更具抵抗力。

## 示例
### 配置示例
在 Nginx 配置 SSL 时指定套件：
```nginx
ssl_ciphers ECDHE-RSA-AES256-GCM-SHA384;
```
