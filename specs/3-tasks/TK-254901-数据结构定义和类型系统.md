## 实现范围

### 1. 核心数据模型

#### 1.1 Session 结构体
```go
// internal/storage/types/session.go
type Session struct {
    // 主键
    SessionID string `json:"session_id"`

    // 核心字段
    UserID     string     `json:"user_id"`
    ClientIP   string     `json:"client_ip"`
    DeviceType DeviceType `json:"device_type"`
    DeviceID   string     `json:"device_id"`
    UserAgent  string     `json:"user_agent"`

    // 会话类型与状态
    SessionType SessionType `json:"session_type"`
    Status      Status      `json:"status"`

    // 时间戳
    CreatedAt     int64 `json:"created_at"`
    LastActiveAt  int64 `json:"last_active_at"`
    ExpiresAt     int64 `json:"expires_at"`

    // 扩展字段
    Metadata       map[string]string `json:"metadata"`
    LocalSessions  []*LocalSession   `json:"local_sessions"`
}

type LocalSession struct {
    System       string `json:"system"`
    LocalID      string `json:"local_id"`
    RegisteredAt int64  `json:"registered_at"`
}
```

#### 1.2 Token 结构体
```go
// internal/storage/types/token.go
type Token struct {
    // 主键
    TokenID   string `json:"token_id"`
    TokenHash string `json:"token_hash"`

    // 关联字段
    SessionID string `json:"session_id"`
    UserID    string `json:"user_id"`

    // 令牌类型
    TokenType TokenType `json:"token_type"`
    Scope     string    `json:"scope"`
    Issuer    string    `json:"issuer"`

    // 时间戳
    IssuedAt  int64  `json:"issued_at"`
    ExpiresAt int64  `json:"expires_at"`
    Status    Status `json:"status"`
}
```

// internal/storage/types/validation.go
func (s *Session) Validate() error {
    if len(s.SessionID) == 0 || len(s.SessionID) > 64 {
        return ErrInvalidSessionID
    }
    if len(s.UserID) == 0 || len(s.UserID) > 128 {
        return ErrInvalidUserID
    }
    if len(s.ClientIP) > 45 {
        return ErrInvalidClientIP
    }
    if len(s.UserAgent) > 2048 {
        return ErrInvalidUserAgent
    }
    // ... 其他验证

}

## 技术要点

1. **字段长度限制**：所有字符串字段必须验证长度
2. **时间戳格式**：统一使用 Unix 纳秒时间戳（int64）
3. **Protobuf 版本**：使用 proto3 语法
4. **YAML 标签**：配置结构需要 `yaml` 标签支持
5. **错误定义**：所有验证错误使用预定义的 sentinel error
