# 想法: 运维管理体系

状态: 已批准

## 原始想法

建立一套完善的运维管理体系，确保 TokMesh 在生产环境下的可维护性和数据安全性。

### 客户端管理工具 (tokmesh-cli)
提供专用的 CLI 工具 `tokmesh-cli`，作为管理员与集群交互的主要入口。
*   **集群管理**: 查看节点状态、手动添加/移除节点。
*   **数据管理**: 触发手动快照、执行数据备份、从备份文件还原。
*   **Key 管理**: 创建、轮转、吊销 API Key。

### 审计日志 (Audit Logging)
*   **覆盖范围**: 所有对系统状态产生变更的操作（如 `Create`, `Update`, `Delete`）及敏感查询。
*   **记录内容**: 操作时间、操作人 (API Key ID)、操作对象、操作结果、来源 IP。
*   **存储**: 审计日志应独立存储或打标，防止被轻易篡改。

### 备份与恢复
*   支持全量备份和增量备份。
*   支持在线热备份，不中断服务。
