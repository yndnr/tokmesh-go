# 想法: 分布式集群架构

状态: 已批准

## 原始想法

TokMesh 支持**去中心化 (Decentralized)** 的分布式集群部署模式。

### 核心特性
1.  **部署灵活性**: 支持单节点模式（开发/测试）和多节点集群模式（生产）。
2.  **对等节点**: 集群中每一个节点都是对等的，均可对外提供读写服务，无单点故障。
3.  **弹性伸缩**:
    *   **快速加入**: 新节点启动后可快速发现并加入集群。
    *   **快速退出**: 节点下线后，集群能感知并重新平衡。
4.  **自侦测**: 节点间具备心跳与状态侦测能力，自动维护集群拓扑视图。
