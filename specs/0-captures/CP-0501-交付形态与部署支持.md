# 想法: 部署支持

状态: 已批准

## 原始想法

TokMesh 需要适配多样化的运行环境，提供开箱即用的**多平台、多架构**交付物，以降低用户的部署和运维门槛。

### 支持平台
1.  **操作系统**:
    *   **Linux**: 提供 `.tar.gz` 压缩包（含二进制文件和 systemd unit 模板），支持主流发行版（Ubuntu, CentOS, Debian 等）。
    *   **Windows**: 提供 `.zip` 压缩包（含 `.exe` 和安装脚本），支持注册为 Windows Service 后台运行。

2.  **CPU 架构**:
    *   **x86_64 (amd64)**: 主流服务器架构。
    *   **arm64**: 适配高性能 ARM 服务器（如 AWS Graviton, 华为鲲鹏）及开发板（如树莓派）。

### 容器化与编排
1.  **Docker / Podman**:
    *   官方维护精简的 OCI 标准镜像（基于 Alpine 或 Distroless）。
    *   支持环境变量配置。
2.  **Kubernetes (K8S)**:
    *   提供官方 **Helm Charts**，简化集群部署。
    *   提供标准 `deployment.yaml`, `statefulset.yaml` 等 K8S 资源清单。
