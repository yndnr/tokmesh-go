# TokMesh 错误代码手册

本文档列出 TokMesh 系统的所有标准错误代码及其含义。

## 错误代码格式

格式：`TM-MMTTSS`

- **TM**：TokMesh 前缀
- **MM**：模块代码（2位数字）
- **TT**：错误类型（2位数字）
- **SS**：序号（2位数字）

**示例**：`TM-020101` = 会话模块(02) + 参数错误(01) + 序号(01)

## 会话模块错误 (TM-02XXXX)

### 参数错误 (TM-0201XX)

| 错误代码 | 标题 | 详细说明 | 建议操作 |
|---------|------|---------|---------|
| **TM-020101** | Session ID 无效 | session_id 长度必须在 1-64 个字符之间 | 请检查 session_id 参数，确保长度符合要求 |
| **TM-020102** | User ID 无效 | user_id 长度必须在 1-128 个字符之间 | 请检查 user_id 参数，确保长度符合要求 |
| **TM-020103** | Client IP 无效 | client_ip 长度不能超过 45 个字符（支持 IPv4 和 IPv6） | 请检查 client_ip 参数格式是否正确 |
| **TM-020104** | User-Agent 过长 | user_agent 长度不能超过 2048 个字符 | 请截断或省略 User-Agent 字符串 |
| **TM-020105** | Device ID 过长 | device_id 长度不能超过 256 个字符 | 请检查 device_id 参数长度 |
| **TM-020106** | Metadata 过大 | metadata 序列化后的大小不能超过 4KB | 请减少 metadata 中的键值对数量或缩短字符串长度 |
| **TM-020107** | 本地会话过多 | local_sessions 数量不能超过 10 个 | 请减少关联的本地会话数量 |
| **TM-020108** | 设备类型无效 | device_type 必须是 web, mobile, desktop, api, iot 之一 | 请检查 device_type 参数值 |
| **TM-020109** | 会话类型无效 | session_type 必须是 normal, vip, admin 之一 | 请检查 session_type 参数值 |
| **TM-020110** | 状态无效 | status 必须是 active, expired, revoked 之一 | 请检查 status 参数值 |

### 未找到错误 (TM-0204XX)

| 错误代码 | 标题 | 详细说明 | 建议操作 |
|---------|------|---------|---------|
| **TM-020401** | 会话未找到 | 指定的 session_id 不存在或已过期 | 请检查 session_id 是否正确，或引导用户重新登录 |

### 业务逻辑错误 (TM-0202XX)

| 错误代码 | 标题 | 详细说明 | 建议操作 |
|---------|------|---------|---------|
| **TM-020201** | 会话已过期 | 会话已超过有效期 | 请引导用户重新登录 |
| **TM-020202** | 会话已吊销 | 会话已被管理员或用户主动吊销 | 请引导用户重新登录 |

### 已存在错误 (TM-0205XX)

| 错误代码 | 标题 | 详细说明 | 建议操作 |
|---------|------|---------|---------|
| **TM-020501** | 会话已存在 | 相同的 session_id 已存在 | 请使用不同的 session_id 或更新现有会话 |

### 资源限制错误 (TM-0209XX)

| 错误代码 | 标题 | 详细说明 | 建议操作 |
|---------|------|---------|---------|
| **TM-020901** | 会话存储已满 | 会话数量已达到系统配置的上限 | 请联系管理员增加存储容量或清理过期会话 |
| **TM-020902** | 会话内存耗尽 | 会话占用的内存已达到系统配置的上限 | 请联系管理员增加内存配额或启用驱逐策略 |

## 错误响应示例

### JSON 格式（用于 API 响应）

```json
{
  "error_code": "TM-020101",
  "title": "Session ID 无效",
  "message": "session_id 长度必须在 1-64 个字符之间",
  "suggestion": "请检查 session_id 参数，确保长度符合要求",
  "details": {
    "field": "session_id",
    "actual_length": "72",
    "max_length": "64"
  }
}
```

### Go 代码使用示例

```go
import "github.com/yndnr/tokmesh-go/pkg/errors"

// 返回标准错误
func ValidateSessionID(sessionID string) error {
    if len(sessionID) > 64 {
        return errors.ErrSessionIDInvalid.
            WithDetails("actual_length", fmt.Sprintf("%d", len(sessionID))).
            WithDetails("max_length", "64")
    }
    return nil
}

// 检查错误类型
if errors.Is(err, errors.ErrSessionNotFound) {
    // 处理会话未找到的情况
}
```
